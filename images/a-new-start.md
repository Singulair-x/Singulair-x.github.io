## 回顾

很早就有建立自己的博客网站的想法和尝试。

初二年级时，我使用某厂商（已停止运营）的免费虚拟主机和来自 [Freenom](https://freenom.com/) （也已不再提供服务[^1]）的免费`.tk`域名，基于 [Typecho](https://typecho.org/) 框架搭建了自己的第一个博客。当时还自己写了一些有趣的小插件，如随昼夜变化的背景图、右下角的看板娘。博客上都发过什么已经记不起来。不过不久也就丧失了写作的兴趣，主机商也很快停运了。

后来的暑假，开始尝试静态网站，边学 HTML、CSS 边手搓，托管在 Github Pages 上。然而我写的样式实在糟糕，观感极差，加上徒手造轮子对于当时的我来说难度还是太大了，最终半途而废。

使用 Github Pages 的经验让我了解到 [Jekyll](https://jekyllcn.com/]) 、Hexo 这样的静态博客框架。后来又相继搭建过 2 个站点，不过也都因为实在没什么可写陆续关停了。<del>难说这个站点是否也会有相同的命运。</del>

## 新的开始

清明节假期突然萌生出建立（又）一个博客的想法，因为有空闲也很快付诸实践。似乎有很多时间花在挑选主题上，最终还是回到了曾经使用过的 Redefine，“没那么简陋而又不至于花哨”。利用大约半天的时间完成各种配置，并配置好了 [Obsidian](https://obsidian.md/) 用于本地写作（详见后续博文）。

**博客的站名**来源于和 AI 的头脑风暴，将一些有特殊含义的文字转成拼音，对字母进行重排，得到一个可读的伪英文单词（规定读作`/ˈzɪz.neɪl/`）。当然，我们可以发挥想象力，编造一下这个词的含意：

> [!summary] AI生成的解释
> - **镜像反转**：如果把“Xiznail”倒着拼写，会得到 **“Lianzi(x)”**。这很容易让人联想到 “莲子”或 “连字（符）”。像是一个在镜子中读出的名字，暗示着这里记录的是“另一个自我”或“真实的内在”。
> - **构词拆解**：
> 	- “Xiz-”：未知、神秘、实验性
> 	- “-nail”：尖锐、固定、刺入
> 	- 组合起来：**钉入现实的异界符号**

博客主题配色起初打算使用清华紫（`#701E5E`），然而该色在暗色模式下效果不佳，转而使用了`#8B5CF6`。亮色和暗色模式的两版背景均使用 Nano Banana 绘制，提示词基于诗句“倚天照海花無數，流水高山心自知”[^2] 。

当前模板似乎解决了 Hexo 默认 Markdown 渲染器和 Mathjax 公式冲突的问题，例如$i\hbar\frac{\partial}{\partial t}\psi=-\frac{\hbar^2}{2m}\nabla^2\psi+V\psi$. 当然，由于其他原因，`hexo-renderer-marked`还是会被卸载掉。

第一篇博文就这样结束。
